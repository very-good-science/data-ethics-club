# How to manage the Reading List with AirTable 

The reading list is currently hosted using the AirTable platform, with the data hosted on an AirTable 'base'. 
If you need to change anything you can log into AirTable using the Data Ethics Club log in detail. 

When a new suggestion is recieved through the suggestions form we will recieve an email on the DEC Slack message. 
In that Slack message you can either approve it immediately using the action button or log into AirTable and tick the box in the column that says 'Display on Website'.
